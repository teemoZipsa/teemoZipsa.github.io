# Teemo Magazine: Auto-Blog Generation Pipeline

이 문서는 티모집사 프로젝트의 AI 에이전트가 "블로그 하나 써줘"라는 명령을 받았을 때 수행해야 하는 공식 자동화 파이프라인(명령서)입니다.
에이전트는 이 파일을 읽은 즉시 아래의 단계별 알고리즘을 100% 무인 자동화로 수행해야 합니다.

## 🤖 런타임 알고리즘 (AI 에이전트 행동 지침)

### Step 1: 타겟 분석 (도구 스캔)
1. `index.html` (또는 `special-chars/` 하위 폴더)을 스캔하여 현재 티모집사 사이트에서 제공 중인 계산기/유틸리티 도구 목록을 파악한다.
2. 기존 `blog/` 디렉토리에 이미 발행된 아티클 목록을 확인한다.
3. **아직 매거진 글과 연결되지 않은(또는 트래픽 푸시가 필요한) 도구 하나를 타겟으로 선정한다.** (예: 아직 글이 없는 '비만도 계산기' 선정)

### Step 2: 최신 소스 스크래핑 (웹 기반 팩트 체크)
1. 선택된 도구와 매칭되는 수의학 주체를 설정한다. (예: Feline Obesity, BCS score)
2. `search_web` 또는 검색 도구를 사용하여 **PubMed Central(PMC), MDPI, JFMS Open Reports** 등 오픈 액세스 저널에서 **2025년 또는 2026년** 최신 무료 논문 원문(결론/초록 포함)을 찾는다.
3. 할루시네이션(거짓 정보)을 방지하기 위해 검색된 논문의 팩트, 통계, P-value 등 구체적 근거를 추출한다.

### Step 3: 원고 작성 (티모집사 프리미엄 톤 & 매너)
1. 검색된 논문의 학술적 내용을 블로그 독자들이 흥미롭게 읽을 수 있도록 번역 및 각색한다.
2. 메인 타이틀을 매력적으로 뽑고, 반드시 `H2`, `H3` 태그 및 `<blockquote>` 태그를 적절히 배치하여 **1,500자 이상의 구글 애드센스 최적화형 원고**를 작성한다.

### Step 4: HTML 패키징 및 위젯 삽입 (Ping-Pong 로직)
1. `blog/article-template.html`의 CSS 및 구조를 그대로 상속받아 새로운 폴더(예: `blog/cat-obesity-guide/index.html`)를 생성하고 글을 삽입한다.
2. 메인 페이지 `index.html` 내의 `MAGAZINE_ARTICLES` 자바스크립트 배열에 새 글 정보를 기입한다.
3. **[매우 중요]** Step 1에서 타겟팅했던 계산기 도구의 `index.html` 하단에 `<div class="article-cta">` 스마트 위젯 코드를 삽입하여, 유저가 방금 작성된 블로그 글로 다시 유입되게 만든다.

## 🚀 사용법 (For User)
새로운 세션을 열었을 때, AI에게 다음 코멘트만 날리세요:
> **"`.agents/workflows/auto_blog_pipeline.md` 매뉴얼대로 블로그 하나 뚝딱 만들어."**
그러면 에이전트가 알아서 스캔, 검색, 작성, 코드 연동까지 100% 진행합니다.
